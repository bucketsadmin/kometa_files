# Kometa collection files
Repo to hold collection based configuration for Kometa. The actual Kometa config file is kept in a private git repo that references this repo.



# Directory Structure
```
kometa_files/
├── movies/
│   ├── collections/
│   │   ├── lists.yml
│   │   ├── manual.yml
│   │   └── smart.yml
│   ├── metadata/
│   │   ├── mediux_movie.yml
│   │   └── movie_edits.yml
│   ├── overlays/
│   │   └── movies.yml
│   └── templates/
│       └── movie_collections
└── tv_shows/
    ├── collections/
    │   ├── lists.yml
    │   ├── manual.yml
    │   └── smart.yml
    ├── metadata/
    │   └── mediux_shows.yml
    ├── overlays/
    │   └── tv_shows.yml
    └── templates/
        └── tv_collections
```
